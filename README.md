1. Top text box: Enter the question you want
2. Send button below: Ask Bard the question I entered when clicking --> You can run the same function without pressing the button and pressing the enter key

***Variable containing the entered string: user_input </br>
--> send user_input to query_optimizer(user_input) function --> return optimized query (= optimized_query) </br></br>

***Variables containing optional query: optional_query </br>
Send an optimized query with --> generate_response(optimized_query) --> return the answer of the bard accordingly (= response)</br></br>

***Variable containing string answered by Bard: response </br>
--> Send response to prompt_optimizer(response) function --> Return response by optimizing (= final result)</br></br>
